## Release model

[Create an OCDS release](../operations/#create-a-release) and:

1. Add 'tender' to `tag`.
1. Set `tender.status` to 'active'.

## What's new

If you have already implemented all prior forms, this notice adds:

* [Preamble](#preamble)
* [III.1.10](#III.1.10) Criteria for the selection of participants
* [IV.1.2](#IV.1.2) Type of contest
* [IV.1.7](#IV.1.7) Names of participants
* [IV.1.9](#IV.1.9) Criteria for the evaluation of projects
* [IV.3](#IV.3) Rewards and jury

Unlike earlier forms, F12 has `/LEFTI/NO_PARTICULAR_PROFESSION`, and the XML schema for `/LEFTI/PARTICULAR_PROFESSION` ([III.2.1](#III.2.1) *Information about a particular profession*) differs. The mapping therefore differs.
